Introducing **Jennah**, the agent memory & context platform that serves as the memory and state layer your agents run against. Jennah unifies short-term execution logs, semantic vector context, and reasoning graphs per agent, with globally distributed, geo-partitioned storage that keeps data close to each agent for ultra-low latency, built-in vector search and in-database inference, and hard multi-tenant isolation — all behind a single, unified API and SDK for spawning agents, writing memory, and running retrieval.

Check it out on [https://jennah.nightblue.io/](https://jennah.nightblue.io/).
